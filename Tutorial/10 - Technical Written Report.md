### Exercise 2:  Compare audience needs 
What differences in purpose, content, and format would you need to consider in the following examples:  
 
1. A report on Internet Privacy Issues for:
    1. a new internet user
        1. Purpose: Raise awareness
        2. Content: Case study and instruction
        3. Format: Correspondent
    2. an e-commerce website developer
        1. Purpose: Close and prevent possible loophole
        2. Content: Technical description of implementation
        3. Format: Instruction, user manual and recommendation

2. Information on Leptospirosis for:
    1. camping enthusiasts
        1. Purpose: Raise awareness
        2. Content: Case study and instruction
        3. Format: Correspondent
    2. restaurant owners
        1. Purpose: Guidelines
        2. Content: Instruction to handle food
        3. Format: Poster
    3. public health department
        1. Purpose: Draft policy for prevention
        2. Content: Long term effect of the disease to the community
        3. Format: Long report

3. Proposal for a new bus shelter design for:
    1. municipal office
        1. Purpose: Propose the new design
        2. Content: Details about the new design
        3. Format: Proposal
    2. newspaper reporters
        1. Purpose: Let public know the new design
        2. Content: Why this new design was used
        3. Format: Long report

Use the schematic in Table 2 to determine what kind of document with what kind of content above would be appropriate for the various audiences below. 

### Table 2. Audience Analysis Schematic

| 4 categories of audiences | Examples or roles                                          | Content and document                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|---------------------------|------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Laypeople                 | General public Non-Specialists General users               | Content: Instruction to handle food Format: Poster This type of document is for a general audience with no specialized knowledge. A poster is a highly effective format for a layperson because it uses simple language, visual aids, and a concise layout to convey critical information quickly. This is ideal for instructions that need to be followed by a wide range of people, such as food safety rules in a common kitchen.                                                                               |
| Executives                | Decision makers – those who manage resources and people    | Content: Long-term effect of the disease to the community  Format: Long report Description: Executives need to understand the big picture. A long report provides the comprehensive, high-level analysis of a complex issue, focusing on its long-term strategic implications. It typically includes an executive summary and data-driven insights that allow them to grasp the full scope of the problem and its impact on the organization or community without getting bogged down in minute technical details. |
| Experts                   | Advisors – who use their expertise to help decision makers | Content: Case study and Instructions Format: Correspondence Audience Needs: Experts need detailed, information to make informed decisions. Purpose: A case study, when presented in a professional correspondence format, provides the specific context and data they need to analyze a problem and implement the instructions. Key Features: The case study serves as a real-world example of how the instructions should be applied, which is valuable for a technical or subject matter expert.                 |
| Technicians               | Implementers                                               | Content: Technical description of implementation Format: Instructions, user manual and recommendation Description: Technicians require highly specific, step-by-step guidance to perform tasks. The technical description provides the necessary detail, while the instructions and user manual formats are ideal for conveying this information clearly. The recommendations offer additional context on best practices, which is useful for someone who will be hands-on with the implementation.                |
